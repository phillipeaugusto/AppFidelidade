
# App Fidelidade


Api criada para validação de conceitos e tem o ituito de gerenciar clientes e seus cartões, juntamente com os fornecedores de produtos / serviços com o segmento de fidelidade/captação de cliente.

- DotnetCore 3.1
- EF
- AutoMapper
- Clean Architecture
- Swagger 
- JWT
- Pagination
- PostgreSQL
